Predict Heart Disease risk with a simple model!

This is not medical advice and should not be used as such! For all medical advice please seek the assitance of a licensed doctor. For education purposes only!!!

The decision tree model was created with a CART (Classification And Regression Tree) algorithm using the Scikit library. It was trained using a dataset from Kaggle.

Even though the dataset contains more attributes than are asked below, it was determined that it performs the best when only the top seven attributes (as determined by the algorithm) were used in training.

More information regarding heart disease and lowering risk can be found at Centers for Disease Control and Prevention.
